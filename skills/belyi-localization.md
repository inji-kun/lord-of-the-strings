description: Enumerate and analyze Belyi maps, covering maps, and moduli space localization
---

<process>
1. Read `config/active-preset.txt`. If it contains a path, Read that preset file.
2. Parse the user's input: graph topology, branching data, or permutation triples (σ_0, σ_1, σ_∞).
3. Read MAPPING_DICTIONARY §V (Belyi maps subsection) for the combinatorial bridge between Feynman diagrams and Belyi maps.
4. Enumerate Belyi maps: holomorphic maps P: Σ_g → CP^1 (compact cover) branched over {0, 1, ∞}.
   In the Gopakumar-Mazenc convention (2212.05999 §8):
   - ∞: Vertex operator insertions create branching with cycle structure (k₁)...(kₙ)
   - 1: The matrix potential (Gaussian: quadratic) creates branching (2)^{|k|/2}
   - 0: Liouville wall at the cigar tip creates the remaining branching
   The three permutations σ₀, σ₁, σ_∞ ∈ S_{|k|} act on DARTS (half-edges), not directly on vertices/edges/faces. Their cycle decompositions encode the combinatorial data. The constraint σ₀·σ₁·σ_∞ = identity reflects trivial monodromy around the three branch points.
   Note: This is specific to the Gopakumar-Mazenc Feynman diagram convention. The standard dessin d'enfants convention uses preimages of 0 and 1 as the two vertex colors of a bipartite graph. The two conventions are related but not identical.
   a. From graph data: extract the permutation triple.
   b. Verify: σ_0 · σ_1 · σ_∞ = identity.
   c. Check transitivity of the group generated by (σ_0, σ_1) — ensures the covering is connected.
5. Compute the degree of the covering map (= number of edges of the graph = number of sheets).
6. Verify the Riemann-Hurwitz formula: 2 - 2g = 2d - Σ (ramification indices - 1). Applied to the compactified cover. The n marked points correspond to branching data, not punctures in this context.
7. Connect to integer Strebel data: the Belyi map defines a point in M_{g,n} corresponding to a Strebel differential with integer edge lengths.
8. If the user provides a dessin d'enfants (bipartite graph on a surface), extract the Belyi data and identify the corresponding Strebel graph.
9. For Hurwitz counting: compute the number of inequivalent Belyi maps with given branching profile.
</process>

- Feynman diagrams ARE Belyi maps (dessins d'enfants), using the Gopakumar-Mazenc convention (2212.05999 §8):
  - ∞: Vertex operator insertions (cycle structure from Tr(M^k) operators)
  - 1: Quadratic potential / Gaussian (cycle structure (2)^{|k|/2})
  - 0: Liouville wall / cigar tip (remaining branching via σ₀·σ₁·σ_∞ = identity)
- The string path integral localizes to discrete points = integer Strebel points.
- The target worldsheet theory is the A-twisted Kazama-Suzuki SL(2,R)/U(1) at k=1.
- For genus 0, the localization mechanism is proven (1911.00378). Higher genus is conjectural.
- Use Hurwitz counting to enumerate covering maps at each genus.
- Corpus pointers: MAPPING_DICTIONARY §V (Belyi maps), 2212.05999 §8, 1911.00378, 0803.2681.
</preset>

<output>

- Verification of covering-map constraints (transitivity, Riemann-Hurwitz).
- Degree and genus of the covering.
- The corresponding point in M_{g,n} (integer Strebel data).
- Hurwitz number / count of inequivalent coverings if requested.
- Connection to the A-model localization program (DSD-III).
</output>
